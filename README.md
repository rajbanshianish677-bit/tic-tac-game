
# 🎮 Tic-Tac-Toe Game

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub repo size](https://img.shields.io/github/repo-size/rajbanshianish677-bit/tic-tac-game)](https://github.com/rajbanshianish677-bit/tic-tac-game)

A responsive Tic-Tac-Toe game built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. 

 This project was created as a coding challenge against  https://github.com/bisugto build the ultimate Tic-Tac-Toe experience.



[🚀 Live Demo](https://rajbanshianish677-bit.github.io/tic-tac-game/)

## ✨ Features

- **Versatile Matchups:** Play locally against a friend or challenge the computer.
- **Undo Move:** Made a mistake? Quickly undo your last move.
- **Score Tracking:** Keep score across multiple rounds.
- **Modern UI:** Clean, minimalist design with smooth interactions.
- **Fully Responsive:** Optimized for both mobile devices and desktop screens.

## 🛠️ Technologies Used

- **HTML5** - Structure and layout
- **CSS3** - Custom styling and responsive design
- **JavaScript (ES6)** - Game logic, state management, and AI behavior

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone [https://github.com/rajbanshianish677-bit/tic-tac-game.git](https://github.com/rajbanshianish677-bit/tic-tac-game.git)

```

2. Navigate into the project directory:
```bash
cd tic-tac-game

```


3. Open `index.html` in your favorite browser.
